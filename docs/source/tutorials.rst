Tutorials
========================================================================

The tutorials presented in this section illustrate different aspects of
THUNER, and how to apply THUNER to different datasets. Most tutorials
are also available as `scripts <https://github.com/THUNER-project/THUNER/blob/main/thuner/test>`__ 
and `notebooks <https://github.com/THUNER-project/THUNER/blob/main/demo>`__ 
in the `THUNER Github repository <https://github.com/THUNER-project/THUNER>`__.
